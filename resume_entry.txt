2. 多智能体代码审查系统（独立开发）
项目描述：针对传统代码审查依赖人工、一致性差、反馈滞后等痛点，基于 LangGraph 多智能体协作架构，从零构建自动化代码审查与缺陷修复系统。设计四角色专业 Agent（静态分析/语义审查/测试评估/自动修复）并行协作管道，集成 ChromaDB 向量知识库实现 RAG 增强审查，搭建 FastAPI 后端与 Web Dashboard 提供完整服务化接口。
主要工作：独立完成系统全栈架构设计与实现，涵盖 LangGraph 状态机编排（9 节点工作流）、Pydantic 数据模型层（20+ 模型）、Semgrep/Pylint/Tree-sitter 工具抽象层（统一四步协议）、RAG 混合检索引擎（ChromaDB + 关键词重排序）、Patch 三重验证管道（语法+静态分析+沙箱执行）及 FastAPI 9 端点 API 服务器，编写 101 个单元/集成测试与 12 样本评估数据集，支持 Mock/DeepSeek 双模式无缝切换。代码仓库：https://github.com/Egor-wang/multi-agent-code-review
